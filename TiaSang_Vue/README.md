

### Prerequisites
- Node.js >= 14.0
- npm hoặc yarn

### Installation
```bash
# Chuyển vào thư mục project
cd TiaSang_Vue

# Cài đặt dependencies
npm install
# hoặc
yarn install
```

### Development Server
```bash
# Chạy dev server
npm run dev
# hoặc
yarn dev
```

Server sẽ chạy tại `http://localhost:5173`

### Build cho Production
```bash
# Build project
npm run build
# hoặc
yarn build
```

Output sẽ nằm trong thư mục `dist/`

### Preview Build
```bash
# Preview build
npm run preview
# hoặc
yarn preview
```
